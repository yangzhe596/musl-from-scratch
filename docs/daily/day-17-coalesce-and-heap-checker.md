# Day 17 - 合并空闲块与调试器

## 今天的目标

- 实现相邻空闲块合并
- 写一个简单的堆检查器
- 提高分配器可调试性

## 预计投入

- 20 分钟：实现合并逻辑
- 20 分钟：写 heap dump/heap check
- 20 分钟：跑边界测试

## 今日任务

1. 实现 `coalesce`
2. 增加辅助调试接口，例如：

```c
void heap_dump(void);
```

3. 测试：
   - 分配 A、B、C
   - 释放 B、C
   - 检查是否合并
4. 记录一个你今天发现的链表或边界 bug

## 重点理解

- 合并的前提通常是“物理相邻”，不是“链表相邻”
- 调试器不是可选项，分配器没有可视化几乎很难调

## 验证标准

- `heap_dump` 输出能帮助你定位块状态
- 合并后能够支持更大的再次分配
